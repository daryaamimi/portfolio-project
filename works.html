Weather App

This is a weather app. I have designed this program with HTML and CSS with the help of Bootstrap. Then, with the help of JavaScript, the program receives the information from the user and finally, with the API, it gives the weather information of any city that the user enters its name. Also, the user can get weather information about his city by clicking on the house icon. Weather information includes exact temperature, description of the situation, maximum temperature, minimum temperature, wind, humidity, visibility and pressure. And this program is able to predict the minimum and maximum temperature of that city in the next 5 days.

Vacation Destination Wish List App

This is a vacation destination wish list app. I designed this program with HTML and CSS. With the help of JavaScript, the program stores the information received from the user in that list. And if the user does not enter the address of any image, an image will be displayed automatically. This information includes the name of the destination, its location, its image and a description about that destination.

Rachel Hollis App

This is an app from the biography of Rachel Hollis. I designed this program with HTML and CSS. Then, with the help of JavaScript, information is taken from the user so that Rachel Hollis (virtually) communicates with the user. Then, if the person is under 18 years old or in Iran, a message will be given that he cannot communicate with that user.
